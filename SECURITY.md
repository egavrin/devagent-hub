# Security Policy

## Reporting a Vulnerability

Please do not open public GitHub issues for security vulnerabilities.

Use GitHub private vulnerability reporting for this repository if it is enabled. If it is not
available, contact the maintainer privately before disclosing details publicly.

## Supported scope

This repository covers:

- workflow orchestration
- persistence and baseline safety
- approval and repair behavior
- GitHub integration used by the validated DevAgent-only path

## Out of scope

- unsupported executor adapters
- target-repo application bugs unrelated to Hub orchestration
- local environment misconfiguration
