# Support

## Usage questions

Open a GitHub issue with enough context to reproduce the operator or workflow problem.

## Bug reports

Use the bug report template for orchestration, persistence, approval, repair, or PR handoff bugs.

## Experimental vs supported

Supported:

- `devagent-hub -> devagent-runner -> devagent execute`
- CLI operator flow with artifact inspection through `status`

Experimental:

- non-DevAgent executor adapters until they have comparable live validation
